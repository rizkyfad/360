Foto 360 Equirectangular

Arahkan kamera, ikuti titik bidik hijau ke dalam kotak, tahan sampai terekam. Output: satu foto equirectangular HD.

Butuh HTTPS + izin kamera & sensor gerak. Di plant: sensor pakai mode relatif (gyro), bukan kompas.

Pengaturan

Naikkan kalau jahitan kurang menyatu, turunkan kalau objek tampak melebar. Default 63°.

Export menyimpan foto mentah + arah tiap titik jadi satu file .json — buka di stitcher.html pada PC untuk dijahit sampai 6K/8K pakai GPU PC. "Jahit di HP" langsung jadi tapi terbatas kemampuan HP.
